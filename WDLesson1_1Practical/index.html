Web Technologies

HTML

HTML is an abbreviation for Hyper Text Markup Language. It is not a programming language, rather it's a markup language. HTML can be used to curate a web page.

CSS

Otherwise known as Cascading Style Sheets is known and used for aesthetics, format, and presentation. It's a styling language that can be used for various elements. It decorates a web page that is made by HTML.

JavaScript

JavaScript is a programming language. It is a high-level, interpreted programming language known for interactive and dynamic content on websites. It creates engaging website elements.
